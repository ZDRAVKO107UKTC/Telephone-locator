1. Основна цел

Кодът предоставя приложение за проследяване на телефонни номера, използвайки:

    Библиотеката phonenumbers за извличане на информация за номера като регион и доставчик на услуги.
    OpenCage Geocoder API за конвертиране на регионална информация в географски координати.
    Tkinter за създаване на графичен потребителски интерфейс, който улеснява взаимодействието с потребителя.

2. Структура на проекта

Проектът е разделен на два основни компонента:

    tracker.py
    Съдържа класа PhoneNumberTracker, който изпълнява основната логика за обработка на телефонните номера, извличане на координати и създаване на карта.

    main.py
    Съдържа класа TrackerApp, който управлява графичния потребителски интерфейс и позволява на потребителя да въведе телефонен номер и да стартира процеса на проследяване.

3. Описание на tracker.py

Класът PhoneNumberTracker включва следните основни методи:

    process_number(number)
    Обработва даден телефонен номер, като извлича информация за региона, часовата зона и доставчика на услуги.

    get_approx_coordinates()
    Използва OpenCage Geocoder API за преобразуване на региона в приблизителни географски координати (ширина и дължина).

    draw_map(phone_number)
    Създава интерактивна карта с помощта на библиотеката folium, като отбелязва местоположението върху картата и я запазва като HTML файл.

4. Описание на main.py

Класът TrackerApp осигурява графичен потребителски интерфейс (GUI) за взаимодействие с потребителя. Основните компоненти са:

    __init__(self, root)
    Създава и настройва основния прозорец на приложението, добавяйки текстови полета, бутони и етикети.

    track_number(self)
    Извиква методите на класа PhoneNumberTracker за обработка на номера, извличане на координати и създаване на карта. Управлява грешки и показва съобщения на потребителя.

5. Как работи

    Потребителят стартира програмата (main.py).
    Въвежда телефонен номер (включително кода на държавата) в текстовото поле на GUI.
    Кликва бутона „Track“. Програмата:
        Обработва телефонния номер.
        Извлича региона и го преобразува в координати.
        Създава карта и я запазва като HTML файл.
    Потребителят вижда съобщение за успех с информация за местоположението и файла с картата.

6. Зависимости

    Python библиотеки:

    pip install phonenumbers folium opencage colorama tkinter

    OpenCage API ключ:
    Регистрирайте се в OpenCage Geocoder и използвайте вашия ключ в кода.

7. Пример за вход и изход

    Вход:
    Телефонен номер: +359888123456 (включително кода на България +359)

    Изход:
        Регион: София, България
        Ширина: 42.6977, Дължина: 23.3219
        HTML файл със създадена карта: +359888123456.html

8. Потенциални грешки и обработка

    Грешка: Въведен е невалиден телефонен номер.
    Решение: Програмата показва съобщение за грешка в GUI и иска нов вход.

    Грешка: Проблем с интернет връзката или API ключа.
    Решение: Уверете се, че интернет връзката работи и API ключът е валиден.
